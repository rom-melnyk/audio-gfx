# Web Audio API demo

## How to run the application:
- **Developement mode:**  
   `npm run dev`
- **Production mode:**
  1. `npm run prod`
  1. Run any server in current directory (for instance, `instant 8080`; make sure it's installed: `npm install -g instant`).
- For **both modes:** open `http://localhost:8080/` in browser.
